Every endpoint is documented in the **Endpoints** section of this tab — an
interactive reference with request/response schemas and a "try it" console,
generated directly from the gateway's route table so it can't drift. Request and
response **contracts** (field-by-field JSON with worked examples) also live in
the [guides](/guides/orders).

<Note>
  Open **Endpoints** in the sidebar to browse every operation with schemas and a
  live request console. Prefer the [Python SDK](/sdks/python) for real
  integrations — it wraps these endpoints with typed models and idempotency.
</Note>

## Endpoint map

All endpoints are under `https://api.trymithril.com/v1` and require a
[bearer key](/authentication).

### Accounts
| Method | Path |
| --- | --- |
| `GET` | `/v1/subaccounts` |
| `POST` | `/v1/subaccounts` |
| `GET` | `/v1/subaccounts/{id}` |
| `DELETE` | `/v1/subaccounts/{id}` |
| `PUT` | `/v1/subaccounts/{id}/credentials` |
| `DELETE` | `/v1/subaccounts/{id}/credentials/{exchange}` |
| `GET` | `/v1/subaccounts/{id}/balances` |
